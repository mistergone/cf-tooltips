Tooltip

The following text is a valid target:

This is sample tooltip content. 
<p>The following text is a valid target:
  <button data-tooltip__target="sample-one" class="tooltip__bare-button"><strong>valid target</strong></button>
</p>
<div class="tooltip__container" data-tooltip__name="sample-one">
  This is sample tooltip content.
</div>
  • By adding the '.tooltip__container' class to an element (most likely a DIV), it becomes a tooltip.
  • A tooltip is further defined by adding the 'data-tooltip__name' attribute to the element. The value of this attribute determines which targets will activate the tooltip.
  • By adding the 'data-tooltip__target' attribute to an element, it becomes a clickable tooltip target, which activates tooltips. The target will activate the tooltip a 'data-tooltip__name' value equal to the target's 'data-tooltip__target' value.
  • Any element is a potential tooltip target, although we strongly recommend interactive elements, such as a BUTTON tag, to make the tooltip targets accessible in the simplest way. Also, for best results, the target should be an inline element, not a block element.
  • Above, we use the '.tooltip__bare-button' class (see below) to 'strip out' the button styles.

Tooltip, Bare Button

The following text is a valid target:

This is sample tooltip content. 
<p>The following text is a valid target:
  <button data-tooltip__target="sample-two" class="tooltip__bare-button">valid target</button>
</p>
<div class="tooltip__container" data-tooltip__name="sample-two">
  This is sample tooltip content.
</div>
  • By adding the '.tooltip__bare-button' class to a BUTTON tag, the border, background, margin, and padding are removed or reset to inherit, thus removing default button styling.
